feat(hf-downloads): Add production-ready Hugging Face downloads dashboard

Implements a complete Hugging Face downloads and metrics dashboard with
real API integration, security hardening, and full accessibility support.

## Features
- Real HF API integration (models, datasets, TTS)
- Secure proxy with token management (Base64 server-side)
- Rate limiting (30 requests/min per IP)
- 4 tabs: Models, Datasets, TTS, Metrics
- Search with sort options (downloads/likes/updated)
- Grid/List view toggle
- Pagination (exactly 10 items per page)
- Secure file downloads via proxy
- Full keyboard navigation and ARIA labels
- Perfect RTL layout for Persian
- Responsive design (mobile to 4K)
- Beautiful UI with gradients and animations

## Security
- Token stored server-side only (Base64 encoded)
- Input validation and sanitization
- Path traversal protection
- CORS configured properly
- Zero token exposure to client

## Accessibility
- WCAG 2.1 AA compliant
- Full keyboard navigation (Tab, Arrow keys)
- ARIA labels on all controls
- Visible focus indicators
- Screen reader compatible
